# Despliegue

## Lovable Cloud

El proyecto se despliega automáticamente en Lovable Cloud.

### URLs

- **Preview**: https://id-preview--8c8fe5e1-414b-456a-81f6-aee39f6c7312.lovable.app
- **Producción**: https://comex-flow-buddy.lovable.app

### Publicar

1. Hacer cambios en el código
2. Verificar en preview
3. Click en "Publish" en Lovable

## Build Manual

```bash
npm run build
```

Los archivos se generan en `dist/`.
